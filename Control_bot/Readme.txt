Описание: 
Telegram-бот, который может удаленно выключить твой ПК или получить скриншот

Применение
Получите копию бота

Запустите консоль и введите следущее: *чтобы установить нужные библиотеки:
1) cd *путь к папке*
2) pip install -r requirements.txt

В файле main.py вставить свой бот-токен

Запускаем main.py

**
Если хочешь, чтобы бот запускался со старта Виндовс, то... немного прогугли ;) и  отредактируй файл bot.vbs
